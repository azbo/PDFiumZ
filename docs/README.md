# Documentation and Resources

This directory contains:
- QR codes for project support (WeChat Pay, Alipay)
- Additional documentation assets

## Adding QR Codes

Place your QR code images here:
- `wechat-pay-qr.png` - WeChat Pay QR code (200x200 or similar)
- `alipay-qr.png` - Alipay QR code (200x200 or similar)

The README.md will automatically display them.
