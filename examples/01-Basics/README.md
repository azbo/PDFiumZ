# 01-Basics - PDFiumZ 基础示例

本目录包含 PDFiumZ 的入门示例，适合初学者。

## 📚 示例列表

### GettingStarted
快速入门示例，演示：
- 初始化 PDFium 库
- 打开 PDF 文档
- 读取基本文档信息
- 正确清理资源

## 🚀 运行示例

```bash
# 编译
dotnet build

# 运行
dotnet run
```

- ✅ 理解基本的文档操作模式
- ✅ 正确的资源管理（using 语句）
